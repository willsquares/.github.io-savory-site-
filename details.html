            <h2 class="text-3xl font-bold text-gray-800 mb-6">Savory Token Summary</h2>

            <p class="text-lg text-gray-700 leading-relaxed">

                Savory is a Solana-based cryptocurrency (<span class="font-bold">$SVRY</span>) designed to transform the restaurant payment landscape. It aims to <span class="font-bold">eliminate traditional credit card fees</span> for restaurants and simultaneously <span class="font-bold">reward diners</span> for using the token. By facilitating direct, fast, and near-zero-fee transactions, Savory creates a mutually beneficial ecosystem where both merchants and customers earn rewards, driving adoption and fostering a more efficient and rewarding dining experience.

            </p>

        </section>


        <section class="w-full max-w-4xl bg-white shadow-lg rounded-xl p-8 mb-12">

                    <h3 class="text-2xl font-semibold text-gray-700 mb-3 flex items-center">

                    </h3>

                    <p class="text-md text-gray-600 leading-relaxed">

                        Customers pay using <span class="font-bold">Savory tokens</span> at participating restaurants, making the process quick and efficient.

                    </p>

                </div>

                <div>

                    <h3 class="text-2xl font-semibold text-gray-700 mb-3 flex items-center">

                        <span class="text-red-500 mr-2">💸</span> Fee Elimination

                    </h3>

                    <p class="text-md text-gray-600 leading-relaxed">

                        Restaurants avoid the typical <span class="font-bold">2-3% credit card processing fees</span>, directly increasing their profit margins.

                    </p>

                </div>

                <div>

                    <h3 class="text-2xl font-semibold text-gray-700 mb-3 flex items-center">

                        <span class="text-yellow-500 mr-2">🎁</span> Dual Rewards

                    </h3>

                    <p class="text-md text-gray-600 leading-relaxed">

                        Both customers and restaurants earn rewards: diners receive <span class="font-bold">cashback</span>, and restaurants get <span class="font-bold">fee rebates</span> in Savory tokens.

                    </p>

                </div>

                <div>

                    <h3 class="text-2xl font-semibold text-gray-700 mb-3 flex items-center">

                    </h3>

                    <p class="text-md text-gray-600 leading-relaxed">

                        Leveraging <span class="font-bold">Solana's robust blockchain</span> ensures <span class="font-bold">fast transactions</span>, <span class="font-bold">near-zero fees</span>, and support for real-time payments via Solana Pay.

                    </p>

                </div>

            </div>

            <div class="grid grid-cols-1 md:grid-cols-2 gap-8">

                <div>

                    <h3 class="text-2xl font-semibold text-gray-700 mb-3 flex items-center">

                        <span class="text-purple-500 mr-2">💡</span> Dynamic Minting via Activity

                    </h3>

                    <p class="text-md text-gray-600 leading-relaxed">

                        New <span class="font-bold">$SVRY tokens are minted dynamically</span> as a direct reward for real economic activity within the ecosystem. This ensures tokens are introduced based on genuine utility.

                    </p>

                </div>

                <div>

                    <h3 class="text-2xl font-semibold text-gray-700 mb-3 flex items-center">

                        <span class="text-orange-500 mr-2">💸</span> Transaction-Triggered Rewards

                    </h3>

                    <p class="text-md text-gray-600 leading-relaxed">

                        When a transaction occurs (e.g., a <span class="font-bold">$50 meal</span>), a smart contract automatically triggers the minting process:

                        <ul class="list-disc list-inside mt-2 text-gray-600">

                            <li><span class="font-bold">1% of the value is minted to the customer</span> as cashback.</li>

                            <li><span class="font-bold">1% is minted to the restaurant</span> as a network rebate.</li>

                        </ul>

                    </p>

                </div>

                <div>

                    <h3 class="text-2xl font-semibold text-gray-700 mb-3 flex items-center">

                        <span class="text-pink-500 mr-2">🔒</span> Controlled Emission

                    </h3>

                    <p class="text-md text-gray-600 leading-relaxed">

                        Reward rates are governed by an <span class="font-bold">emission schedule</span>, with <span class="font-bold">inflation capped</span> and gradually reduced over time to maintain token stability.

                    </p>

                </div>

                <div>

                    <h3 class="text-2xl font-semibold text-gray-700 mb-3 flex items-center">

                    <p class="text-md text-gray-600 leading-relaxed">

                        Future enhancements may include <span class="font-bold">staking mechanisms</span> for token holders, <span class="font-bold">burn mechanics</span> to reduce supply, and special <span class="font-bold">merchant bonuses</span> to incentivize adoption.
